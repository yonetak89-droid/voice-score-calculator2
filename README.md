# 5発スコア v5

ピストル競技向けの、Safari対応音声スコア計算Webアプリです。

## v5の変更点

- 5発目の入力後も、5発の点数と計算結果を現在欄に保持
- 次の点数（6発目）が入力された瞬間に、前の5発を履歴へ移動
- 前の結果を履歴へ移した後、その入力を新しい5発の1発目として登録
- 音声で複数点数を連続入力した場合も、5発の境界をまたいで同じ規則で処理
- 完了済みの現在欄に対して「1つ戻す」「現在欄をクリア」が可能
- v4の入力途中データ・履歴・言語設定を引き継ぐ

## 動作例

1. `10・9・8・X・7`を入力
2. 現在欄に5発と`44（X = 1）`を保持
3. 次に`6`を入力
4. `10・9・8・X・7`が履歴へ移動
5. 現在欄は`6`から新しく開始

## 認識ルール

以下はすべて `10` として扱います。

- じゅう
- じゅう点
- じゅうてん
- じゅってん
- 10点
- １０点
- 十点

Xとして扱うのは「インナー」「インナーテン」「インナー10」です。
「エックス」や文字のXはXとして扱いません。
