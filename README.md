# 音声スコア計算アプリ v3（高速入力版）

## 今回の修正
- 音声認識の中間結果を利用し、最終確定を待たず約180msの安定待ち後に入力
- 中間結果と最終結果による重複登録を防止
- `10点`、`１０点`、`十点`、`じゅう点`、`じゅってん`を10として処理
- 全角数字を半角へ正規化
- Service Workerのキャッシュをv3へ更新

## 主な機能
- 0〜10、日本語の「インナー」「インナーテン」を音声入力
- 英語モードではzero〜ten、inner、inner tenに対応
- Xは10点として加算し、X数を別表示
- 5発で自動確定し、計算過程と `合計（X = 個数）` を表示
- 手動入力、1つ戻す、入力クリア、履歴保存・削除

## GitHub Pagesへの更新
このフォルダ内の4ファイルを、既存リポジトリの同名ファイルへ上書きしてください。

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `README.md`

更新後も旧版が表示される場合は、Safariのページを再読み込みしてください。ホーム画面版は一度終了して開き直します。それでも変わらない場合は、SafariのWebサイトデータから該当サイトを削除して再アクセスしてください。

## 注意
SafariのWeb Speech APIは端末・通信・周囲の音に依存します。今回の修正は待ち時間を大幅に短縮する設計ですが、常に1秒未満を保証するものではありません。
