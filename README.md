# 5発スコア v6

## 変更点
- 音声認識を日本語（ja-JP）に統一
- 英語モード・言語切替を廃止
- カタカナ英語を数字として認識
  - ゼロ、ワン、ツー、スリー、フォー、ファイブ
  - シックス、セブン、エイト、ナイン、テン
- 従来の日本語数字、10点、じゅってん、インナー、インナーテンも継続対応
- 5発完了後は結果を現在欄に保持し、次の点数が入った時点で履歴へ移動

## GitHub Pagesへの更新
ZIPを展開し、以下を既存リポジトリへ上書きしてください。
- index.html
- manifest.webmanifest
- sw.js
- README.md

反映後、Safariでページを再読み込みしてください。旧版が残る場合は、ホーム画面版を終了し、SafariからURLを開き直してください。
