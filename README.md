# 5発スコア v4

ピストル競技向けの、Safari対応音声スコア計算Webアプリです。

## v4で見直した点

- 連続発話の処理を修正し、同じ認識区間に後から追加された点数も取り込む
- 中間認識を約240ms安定した時点で採用し、最終確定待ちを短縮
- `10点`、`１０点`、`十点`、`じゅう点`、`じゅうてん`、`じゅってん`を10として処理
- `インナー10`、`インナーテン`もXとして処理
- 「エックス」や文字のXはXとして扱わない
- 入力途中の点数・履歴・言語設定を端末内へ保存
- 音声入力中は対応端末で画面スリープを抑制
- 権限拒否時の自動再起動ループを防止
- 未使用処理と重複表示を削除し、画面を簡素化
- PWAアイコン、オフライン表示、更新時のキャッシュ刷新を追加

## 認識例

以下はすべて `10` です。

- じゅう
- じゅう点
- じゅうてん
- じゅってん
- 10点
- １０点
- 十点

Xとして扱うのは次だけです。

- インナー
- インナーテン
- インナー10
- Englishモードの inner / inner ten / inner 10

## GitHub Pagesへの更新

既存リポジトリへ、このフォルダ内のファイルをすべてアップロードして同名ファイルを上書きしてください。

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
- `README.md`

GitHub Pages反映後、Safariでページを再読み込みしてください。ホーム画面版が旧表示のままなら、一度終了してSafariからURLを開き直してください。

## 制約

Safariの音声認識速度は、通信状態・周囲の音・iOS側の認識処理に左右されます。アプリ側では中間結果を早期採用しますが、常時1秒未満を保証するものではありません。
