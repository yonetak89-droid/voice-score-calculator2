# 音声スコア計算アプリ

## 主な機能
- 0〜10の音声入力
- 「インナー」「インナーテン」をXとして認識
- Xは10点として加算し、X数を別途表示
- 0も1発としてカウント
- 5発で自動確定
- 計算過程と `合計（X = 個数）` を表示
- 1つ戻す
- 入力中の数字をクリア
- 履歴の個別削除
- 履歴の全削除
- 履歴をブラウザ内に保存
- 手動入力
- ホーム画面追加向けPWA構成

## 起動方法
音声認識は、HTTPSまたはlocalhost上での利用を推奨します。

### パソコンで簡易起動
このフォルダ内で次を実行します。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

### iPhoneで使う
Webサーバーへ配置し、SafariでURLを開いてください。
共有メニューから「ホーム画面に追加」を選ぶとアプリ風に利用できます。

## 注意
- 音声認識の可否や挙動はブラウザ環境に依存します。
- 「エックス」またはアルファベットのXは、仕様どおりXとして認識しません。
- 認識できない場合でも手動入力を利用できます。
